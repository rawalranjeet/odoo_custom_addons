OVERVIEW

Odoo - Magento 2 Connector transmits and syncs data from Odoo to the Magento system. This extension automates all the Magento Odoo Integration operation that saves a lot of time and effort without manual errors.

HIGHLIGHT FEATURES

  1. Odoo Magento 2 Integration

  • Error-free data transfer automation between Odoo and Magento 2
  • Manage entire operations directly from Odoo.

  1. Product Management

  • In Magento 2: Automatically update product information (product name, price, check and stock quantity).

  • In Odoo: Set up Product Images, Short Descriptions, Price, Magento Attribute Sets, and Extra Images.

  1. Inventory

  • Update inventory data through two-way synchronization.

  • Update stock in Magento 2 stores according to inventory changes to locations in Odoo

  1. Order Management

With the addition of Odoo Integration for Magento 2 by Magenest, users can import orders from Odoo to Magento 2  and vice versa.

 

  1. Shipment

Update Shipment Details from Odoo to Magento 2 automatically.

1. Access Magento Menu, Configure and Create New Magento Instance

2. Add parameters in instance form including Name, Magento URL, Access token, Prefix for Sale Order

3. Fetch Attribute Set ( If wanna  Synchronize Product)
